# Basic Calculator 

A cute pastel-themed calculator UI inspired by retro CASIO calculators and designed in Figma. Built using HTML, CSS, and Tailwind CSS utilities.

## Live LInk


## Features

* Responsive calculator layout
* Custom pastel color palette
* Retro-style button shadows
* Press animation for buttons
* Floating kawaii cat mascot
* Special operator buttons
* Double-height "=" button
* Mobile-friendly design

## Built With

* HTML5
* CSS3
* Tailwind CSS
* Google Fonts (Luckiest Guy)
